# Android

Auto-generated Android directory for Zach's personal website.

## Commands

### Project Root

Run these from project root, not `/android`!

```bash
# Sync
npm run cap:sync
```

```bash
# Start (for development / testing purposes in emulator)
npm run cap:start
```

## Building

1. Open Android Studio.
2. In the top tool bar, click on `Build > Generate App Bundles or APKs > Generate APKs`.
3. Move the built APK onto your phone, install it (you may have to configure your settings to install apps from "unsupported" sources).
