Pi-Weather

Project Repo

Utilized Svelte to create a weather dashboard, initially with the goal of it being utilized on with a Raspberry Pi to create a physical weather dashboard. However, a bug in the Chromium browser is inserting a white rectangle at the bottom of the page when screenshotting, preventing it from screenshotting properly. I will probably attempt to remake this project using a templating framework over a web framework. This project was a good introduction to APIs and Svelte.

Replace Text Python Package

Project Repo PyPi Page

Description

  • In-file text replacer developed using Python.
  • Can replace text in a single file, or in every file within a specified directory.
  • Supports various flags, speeding up the process of using the CLI.
