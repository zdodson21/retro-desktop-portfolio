Py Weather

Project Repository

Description:

Inspired by a similar project by AKZ DEV I utilized Python combined with the Jinja templating engine to create a weather dashboard. This dashboard is designed to run on Raspberry Pi computers to enable users to create their own home-made weather displays alongside being simple to modify so users can use their preferred (supported) language and units of measurement.

Details:

Software Development

  • Ported this project from Svelte to Python to improve usability. Porting to Python enabled this project to be more efficient, run without a local server, and run natively on Raspberry Pi with minimal installs needed.
  • Utilized Python combined with the Jinja templating engine to call the Open Weather One Call API, Air Quality API, and Geocoding API. Data is decoded into human readable values, which are populated into the template.html.j2 file.
  • Created a prototype user-interface using the Penpot design software, then utilized the SASS style sheet language to improve developer experience of modifying the user interface while supporting various screen sizes.

Raspberry Pi / Linux Configuration

  • Combined Bash and Python scripting to ensure program can be started natively or via SSH
    • Bash: Used for anything relating to the system, such as ensuring variables are correct, requiring the user to use the Python virtual environment, displaying the dashboard image, calling Python scripts on a set timer (cron), and more.
    • Python: Used for populating an HTML file with data filled in from the Open Weather APIs and obtaining a screenshot of the rendered dashboard.
