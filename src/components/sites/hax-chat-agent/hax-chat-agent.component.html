HAXTheWeb Chat Agent Frontend (Internship Project) (May 2024 - August 2024)

Source Code HAXTheWeb Website Lit Library

Description:

Between May 2024 - August 2024 I created numerous web components for the HAXTheWeb project at the Penn State College of Arts & Architecture. These web components were utilized to create one main component called chat-agent. The goal of this project was to create a chat-agent for websites created within the HAX Content Management System (HAXCMS) that could act as a "virtual learning assistant". This chat agent (named Merlin) would utilize site content as its context to answer student questions about course material, while also being capable of answer questions about HAXCMS (for developers). These components work together to create a seamless experience for both end-users and developers.
