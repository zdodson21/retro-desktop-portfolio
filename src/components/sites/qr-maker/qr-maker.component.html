QR Maker

Project Repository

Description

  • Developed a QR code maker application using the Flutter framework and Dart programming language to target various platforms with one codebase.
  • Users can save QR code generating strings to their device, enabling them to quickly generate QR codes from their saved options. One use case is to quickly share multiple hyperlinks to different pages on a website.

Known Issues

Because I would like to move onto other projects, these issues are known but will not be fixed soon. I will likely fix them at some point in the future, but I have other projects I would like to work on for the time being.

  • The Windows build of this application returns an error when trying to save QR code as image. This is likely a permissions issue, but will not be fixed soon due to Windows not being a primary build target.
  • The Android build of this application was also returning an error when trying to save QR code as an image. Again, this is likely caused by a permissions issue. For now the button is disabled on Android devices (mobile version is intended to be used to display QR codes using the "Select" menu).
