QR Maker

Project Repository

Note: Work in progress. Linux version works but is not feature complete. Android and Windows versions are WIP.

Description

  • Developed a QR code maker application using the Flutter framework and Dart programming language to target various platforms with one codebase.
  • Users can save QR code generating strings to their device, enabling them to quickly generate QR codes from their saved options. One use case is to quickly share multiple hyperlinks to different pages on a website.
